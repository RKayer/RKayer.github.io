print()函数

print()

相当于C语言中的printf()，但是()内的内容可以为变量如print(a),也可以为字符串print(‘Hello Python’)，数字打印print(123)，无须单独指明数据的类型，只需要采用正确的格式即可。

基本数据类型

一，字符串

普通字符串’string’or’’string’’or’’’string’’’都表示一个字符串string，但是三个用法还是有区别的。
比如要字符串里要显示’’，必须这样’’str’i’ng’’依次类推。另外’’’可以分行：

‘’’s
t
r’’’

打印出来就是

s
t
r

二，整型int，浮点型float，布尔值bool（True or False)

Python中的运算符+ - * 和C语言相同

/ 结果为浮点数； // 结果为向下取整

3 / 1 –> 1.0
3 / 2 –> 1.5
3 // 2 –> 1
-3 // 2 –> -2 # 向负无穷方向取整
-3 / 2 –> -1.5

三，基本数据类型转换

方法 说明
int(x [,base ]) 将x转换为一个整数
float(x ) 将x转换到一个浮点数
complex(real [,imag ]) 创建一个复数
str(x ) 将对象 x 转换为字符串
repr(x ) 将对象 x 转换为表达式字符串
eval(str ) 用来计算在字符串中的有效 Python 表达式,并返回一个对象
tuple(s ) 将序列 s 转换为一个元组
list(s ) 将序列 s 转换为一个列表
chr(x ) 将一个整数转换为一个字符
unichr(x ) 将一个整数转换为 Unicode 字符
ord(x ) 将一个字符转换为它的整数值
hex(x ) 将一个整数转换为一个十六进制字符串
oct(x ) 将一个整数转换为一个八进制字符串

Python的变量

Python中的变量类型是不固定的，也就是说一个变量可以赋予它任何类型的数据，就算其已经被定义。

a = 3
a = ‘string’
print(a)

结果输出为string

变量赋值

可以同时多个变量赋值

a = b = c = 1

也可以这样分别赋值

a , b , c = 1 , ‘23’ , 12

列表

List列表表示一系列数据的集合

创建列表

list_name = [1 , 'Python' , 234]

列表中的每个元素类型可以不同

list_name = [123 , 'zhuguang' , 234]
print(list_name)

结果

[123 , ‘zhuguang’ , 234]

列表的访问

可以通过列表的下标索引来访问列表中的值，同样你也可以使用方括号的形式截取数据

列表的索引可以从左至右 ，也可以从右到左 

[123 , 'zhuguang' , 234]
 0         1         2

-3        -2         -1

比如对上面的列表

print(list_name[0])
print(list_name[-1])
print(list_name[0:2])

输出为

123
234
[123 , ‘zhuguang’]

列表的数据是可修改的,直接这样修改就可以

list_name[0] = 12  # 修改数据项
list_name.append(345)  # 添加项到列表末尾
del list_name[1]    # 删除项

列表的运算

Python 表达式 结果 描述
len([1, 2, 3]) 3 计算元素个数
[1, 2, 3] + [4, 5, 6] [1, 2, 3, 4, 5, 6] 组合
[‘Hi!’] * 4 [‘Hi!’, ‘Hi!’, ‘Hi!’, ‘Hi!’] 复制
3 in [1, 2, 3] True 元素是否存在于列表中
for x in [1, 2, 3]: print x, 1 2 3 迭代

元组

元组tuple相对于列表最大的区别就是其不可变，数据一旦赋值就不可修改

创建元组

tuple_name = (123 , 'zhuguang' , 234)
tuple_name = (123,)  # 元组只有一个元素需要在末尾添加逗号

元组的访问与列表的访问相同

元组虽然不能进行元素修改但是可以进行元组的连接等等以及删除整个元组

print(tuple_name * 2)
print(tuple1 + tuple2)
del tuple_name

元组内置函数

方法 描述
len(tuple) 计算元组元素个数
max(tuple) 返回元组中元素最大值（不同类型的可能不能比较，比如整型和字符串）
min(tuple) 返回元组中元素最小值
tuple(seq) 将列表转换为元组

字典

字典可以理解为为序列中每个元素单独赋予了一个名字来方便我们去访问

字典的创建

dict_name = &#123;'key1_name': 12 , 'key2_name' : 'zhuguang','key3_name' : 234&#125;
print(dict_name['key1_name'])  # 字典元素的访问
dict_name['key_name'] = 23  ----  # 如果key_name在dict中就视为修改键值，若不存在视为添加元素

字典的键值可以为任何数据类型

set

set是一个无序不重复元素集

list1 = [12 , 23 , 'zhuguang' , 12 , 23]
set1 = set(list1)

输出为

{‘zhuguang’, 12, 23}

set输入为列表，输出为字典，同时自动去除重复的值，且输出是无序的，并不会按照原来的排序方式

set1.add(22)—-#添加元素
set1.remove(12)—–#删除元素

函数的参数

函数的参数有三种类型，默认参数，关键字参数（位置参数），不定长参数

1.默认参数

默认参数指定义函数时设置一个默认值，在调用函数时如果不对参数进行定义，那么就会使用默认值，默认参数只能在形参表最后一个。

def print_user_info( name , age , sex = '男' ):
    # 打印用户信息
    print('昵称：&#123;&#125;'.format(name) , end = ' ')
    print('年龄：&#123;&#125;'.format(age) , end = ' ')
    print('性别：&#123;&#125;'.format(sex))
    return

# 调用 print_user_info 函数

print_user_info( 'xiaohong' , 18 , '女')
print_user_info( 'zhuguang' , 19 )

2.关键字参数（位置参数）

调用函数对参数进行赋值时使用关键字，且不同关键字的顺序可以与定义函数时的顺序不同。

# 调用 print_user_info 函数

print_user_info( name = 'xiaohong' ,age = 18 ,sex = '女')

3.不定长参数

在函数定义时使用*name表示该参数不定长，那么在调用时可以赋给任意长度的值。

def print_user_info( name ,  age  , sex = '男' , * hobby):
    # 打印用户信息
    print('昵称：&#123;&#125;'.format(name) , end = ' ')
    print('年龄：&#123;&#125;'.format(age) , end = ' ')
    print('性别：&#123;&#125;'.format(sex) ,end = ' ' )
    print('爱好：&#123;&#125;'.format(hobby))
    return;

# 调用 print_user_info 函数
print_user_info( 'zhuguang' ,19 , '男', '打篮球','打羽毛球','跑步')

输出结果

昵称：zhuguang 年龄：19 性别：男 爱好：('打篮球', '打羽毛球', '跑步')

可以看到在赋值时直接赋给hobby多个值，最后输出是以元组输出的。

可变长参数也支持关键字参数，**hobby就可以就表示关键字参数。

只接受关键字参数

在形参表列中在指定元素只能进行关键字参数的前面加一个*元素，其后所有的元素只能进行关键字参数。

( name , * ， age , sex = ‘男’ , * hobby)
